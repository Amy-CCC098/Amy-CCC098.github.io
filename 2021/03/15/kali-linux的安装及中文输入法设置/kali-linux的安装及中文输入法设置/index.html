kali linux的安装及中文输入法设置

渗透必备——vmware安装kali linux及设置中文输入法

kali linux是基于Debian的Linux发行版，里面预装了很多渗透测试的必备软件，比如namp,burpsuite,wireshark等。同时还有基于ARM架构的镜像，可以安装在硬盘、usb驱动器、树莓派、VMware等，这里主要介绍将kali linux安装至VMware的步骤。

安装 kali linux

  1. kali官网：https://www.kali.org
  2. 找到对应版本的种子文件，并下载。
  3. 打开VMware，创建一个新的虚拟机，在映像文件处选择刚刚下载的.iso文件。
  4. 点击下一步，操作系统选Linux，版本为Debian 8.x 64位
  5. 点击下一步，给虚拟机起一个名，选择存放的位置
  6. 点击下一步，选择磁盘大小
  7. 点击下一步，完成

第一次开启kali

  1. BIOS mode选择Graphical install
  2. 语言选择中文简体，国家中国，键盘映射位汉语，然后等待配置
  3. 输入主机名-继续
  4. 输入用户名，以及密码（密码要记住，以后开机以及权限提升的时候都需要用到），等待一段时间配置
  5. 磁盘分区选第一个，不停继续，当跳出将改动写入磁盘吗？，选择“是”，等待一段时间
  6. 接着会跳出软件选择的页面，GNPME和KDE Plasma可以不选，如果做渗透其他尽量选上，等待一段时间
  7. 将GRUB启动引导器安装至您的主驱动器选择“是”
  8. 安装启动引导器的设备选择“/dev/sda”，等待一段时间，安装完成。

设置中文输入法

安装完kali后会发现输入法只能使用英文，所以需要我们自行安装中文输入法

  1. su root获取管理员权限

  2. apt-get install ibus-pinyin安装相应的输入法

  3. ibus首选项将pinyin添加进去

  4. 命令行输入im-conifg进入输入法设置

  5. 选择ibus输入法

    （类似的设置网上有很多，涉及不同的输入法）
