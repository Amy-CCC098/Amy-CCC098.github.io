二、基础语言

标识符、对象、运算符

  • 不能以数字开头
  • 避免和关键字相同（如if,for等）
  • 避免和预定义标识符相同（如int,float等）
  • 大写、小写不一样
  • 双下划线也需要尽量避免使用
运算符 意义
+，-，*，/ 加减乘除
// 整除
** 幂运算
% 取模
<<、>> 左移、右移一位
& 按位与
| 按位或
^ 按位异或
==、!= 相等、不相等
= 赋值
and、or 和、或

python书写规范

  • 一句一行
  • 注意缩进
  • 反斜杠“\”用于一个代码跨多行（字符串、元组、列表、字典可以不需要反斜杠跨多行）
  • 分号“；”用于一行写多个语句
  • 单行注释用#号
  • 多行用’’’…….’’’

元组

元组，可以用一个变量（对象）来存储多个数据，和列表的区别是元组的元素不能修改。元组虽然不能修改，但在创建时间和占用的空间上面都优于列表。

字典

是一个可变容器，它可以存储任意类型对象，与列表、集合不同的是，字典的每个元素都是由一个键和一个值组成的“键值对”，键和值通过冒号分开。
